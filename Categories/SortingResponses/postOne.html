Greetings Paul,
While there are a lot of sorting methods in use today, we will mainly focus on the following sorting methods for this class:
 - selection sort
 - bubble sort
 - insertion sort
 - Shell sort
 - merge sort
 - heapsort
 - quicksort

Hope this helps!
